## 1. 關於
一個專為指揮官而設的工具集。包括圖片庫、製造時間 (人形，裝備，妖精)、圖鑑 (人形，妖精，晶片，重裝)、模擬製作 (人形，裝備，晶片組合)、Line bot 教學頁面及好友專區。
